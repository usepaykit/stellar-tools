Webhooks send HTTP POST requests to your URL when events occur (e.g. `payment.confirmed`, `subscription.canceled`).

## Events

| Event | Description |
|-------|-------------|
| `customer.created` | New customer created |
| `customer.updated` | Customer updated |
| `customer.deleted` | Customer deleted |
| `checkout.created` | Checkout created |
| `payment.pending` | Payment submitted, awaiting confirmation |
| `payment.confirmed` | Payment confirmed on-chain |
| `payment.failed` | Payment failed |
| `refund.succeeded` | Refund completed |
| `refund.failed` | Refund failed |
| `subscription.created` | Subscription created |
| `subscription.updated` | Subscription updated |
| `subscription.canceled` | Subscription canceled |

## Create webhook

If `secret` is omitted and you use a session token, a secret is auto-generated. Use the returned `secret` to verify signatures. Response `200`: `{ "data": { "id", "url", "events", "secret", "name", "description", "isDisabled", ... } }`

<CodeGroup>

```typescript SDK
const webhook = await stellar.webhooks.create({
  name: "Production webhooks",
  url: "https://yoursite.com/webhooks/stellar",
  events: ["payment.confirmed", "subscription.canceled"],
  description: "Optional description",
  secret: "optional_custom_secret",
});
```

```bash cURL
curl -X POST "https://stellartools.dev/api/webhooks" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Production webhooks",
    "url": "https://yoursite.com/webhooks/stellar",
    "events": ["payment.confirmed", "subscription.canceled"],
    "description": "Optional description",
    "secret": "optional_custom_secret"
  }'
```

</CodeGroup>

## Get webhook

Response `200`: `{ "data": { webhook object } }`

<CodeGroup>

```typescript SDK
const webhook = await stellar.webhooks.retrieve("whsec_xxx");
```

```bash cURL
curl -X GET "https://stellartools.dev/api/webhooks/whsec_xxx" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>

## Update webhook

Response `200`: `{ "data": { updated webhook } }`

<CodeGroup>

```typescript SDK
const updated = await stellar.webhooks.update("whsec_xxx", {
  name: "Updated name",
  url: "https://yoursite.com/webhooks/new",
  events: ["payment.confirmed", "refund.succeeded"],
  description: "Optional",
  isDisabled: false,
});
```

```bash cURL
curl -X PUT "https://stellartools.dev/api/webhooks/whsec_xxx" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated name",
    "url": "https://yoursite.com/webhooks/new",
    "events": ["payment.confirmed", "refund.succeeded"],
    "description": "Optional",
    "isDisabled": false
  }'
```

</CodeGroup>

## Delete webhook

Response `200`: `{ "data": { "success": true } }`

<CodeGroup>

```typescript SDK
await stellar.webhooks.delete("whsec_xxx");
```

```bash cURL
curl -X DELETE "https://stellartools.dev/api/webhooks/whsec_xxx" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>
