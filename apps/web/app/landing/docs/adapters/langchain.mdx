---
title: 'LangChain Adapter'
description: 'Meter LangChain model usage with Stellar Tools'
---

The LangChain adapter wraps a LangChain language model and meters usage (tokens) via Stellar Tools credits. It uses `createMeteredPlugin` from `@stellartools/plugin-sdk` for preflight, charge, and refund.

## Install

```bash
pnpm add langchain-adapter @stellartools/plugin-sdk @langchain/core langchain
```

## Usage

Use `createMeteredModel(config, baseModel)` to get a metered model. Pass `customerId` as the first argument to each method; remaining arguments match the base model’s `invoke`, `stream`, `batch`, and `streamEvents`.

```ts
import { createMeteredModel } from "langchain-adapter";
import { ChatOpenAI } from "@langchain/openai";

const config = {
  apiKey: process.env.STELLAR_TOOLS_API_KEY!,
  productId: "prod_llm",
};

const baseModel = new ChatOpenAI({ model: "gpt-4o", temperature: 0 });
const meteredModel = createMeteredModel(config, baseModel);

// customerId is the first argument; then the same args as the base model
const result = await meteredModel.invoke("cust_xxx", [{ role: "user", content: "Hello" }]);

// Streaming: customerId first, then messages and options
for await (const chunk of meteredModel.stream("cust_xxx", [messages])) {
  process.stdout.write(chunk.content);
}

// Batch and streamEvents also take customerId as first argument
const results = await meteredModel.batch("cust_xxx", [messages]);
```

Methods:

- `invoke(customerId, input, options?)` — preflight, run, then charge by token usage.
- `stream(customerId, input, options?)` — preflight, stream, then charge by usage from chunks.
- `batch(customerId, inputs, options?)` — preflight, batch run, then charge by total usage.
- `streamEvents(customerId, input, options?)` — preflight, stream events, then charge by usage from `on_chat_model_end` / `on_llm_end`.

## Config

`createMeteredModel(config, baseModel)` expects:

- `config`: `MeteredPluginConfig` from `@stellartools/plugin-sdk` (`apiKey`, `productId`).
- `baseModel`: Any LangChain `BaseLanguageModel` (e.g. `ChatOpenAI`, `ChatAnthropic`).

Use a Stellar Tools credit product; if the customer has insufficient balance, preflight fails and the call is not made.
