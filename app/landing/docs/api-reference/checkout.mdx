Checkouts represent a single payment or subscription flow. Create a checkout to get a `paymentUrl` or payment URI for your customer.

## Create checkout

Product checkout requires `productId`. Direct checkout requires `amount` and `assetCode` (e.g. XLM, USDC). Response `200`: `{ "data": { "id", "paymentUrl", "status", "expiresAt", "customerId", ... } }`

<CodeGroup>

```typescript SDK
import { StellarTools } from "@stellartools/core";

const stellar = new StellarTools({
  apiKey: process.env.STELLAR_TOOLS_API_KEY!,
});

// Product checkout (amount from product)
const checkout = await stellar.checkouts.create({
  productId: "prod_xxx",
  customerEmail: "buyer@example.com",
  customerPhone: "+1234567890",
  description: "Order #1234",
  successUrl: "https://yoursite.com/success",
  successMessage: "Payment received!",
  metadata: {},
});

// Direct checkout (custom amount)
const directCheckout = await stellar.checkouts.createDirect({
  amount: 10.5,
  assetCode: "XLM",
  customerEmail: "buyer@example.com",
  description: "Donation",
  successUrl: "https://yoursite.com/thanks",
});
```

```bash cURL
# Product checkout
curl -X POST "https://stellartools.dev/api/checkout?type=product" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "productId": "prod_xxx",
    "customerEmail": "buyer@example.com",
    "customerPhone": "+1234567890",
    "description": "Order #1234",
    "successUrl": "https://yoursite.com/success",
    "successMessage": "Payment received!",
    "metadata": {}
  }'

# Direct checkout
curl -X POST "https://stellartools.dev/api/checkout?type=direct" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10.5,
    "assetCode": "XLM",
    "customerEmail": "buyer@example.com",
    "description": "Donation",
    "successUrl": "https://yoursite.com/thanks"
  }'
```

</CodeGroup>

## Get checkout

Response `200`: `{ "data": { checkout object } }`

<CodeGroup>

```typescript SDK
const checkout = await stellar.checkouts.retrieve("cz_xxx");
```

```bash cURL
curl -X GET "https://stellartools.dev/api/checkout/cz_xxx" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>

## Update checkout

Response `200`: `{ "data": { updated checkout } }`

<CodeGroup>

```typescript SDK
const updated = await stellar.checkouts.update("cz_xxx", {
  status: "open",
  metadata: { orderId: "ord_123" },
});
```

```bash cURL
curl -X PUT "https://stellartools.dev/api/checkout/cz_xxx" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "open",
    "metadata": { "orderId": "ord_123" }
  }'
```

</CodeGroup>

## Get checkout status (public)

Used by the payment page or client to poll payment status. No API key required (CORS enabled). Response `200`: `{ "status": "pending" | "completed" | "expired" | "failed" }`

<CodeGroup>

```typescript SDK
const res = await fetch("https://stellartools.dev/api/checkout/cz_xxx/status");
const { status } = await res.json();
```

```bash cURL
curl -X GET "https://stellartools.dev/api/checkout/cz_xxx/status"
```

</CodeGroup>

## Get payment URI

Returns a Stellar `web+stellar:tx` URI for wallet redirect or QR. Requires `environment` query param (`testnet` or `mainnet`). Response `200`: `{ "uri": "web+stellar:tx?..." }`

<CodeGroup>

```typescript SDK
const res = await fetch(
  "https://stellartools.dev/api/checkout/cz_xxx/uri?environment=testnet"
);
const { uri } = await res.json();
```

```bash cURL
curl -X GET "https://stellartools.dev/api/checkout/cz_xxx/uri?environment=testnet"
```

</CodeGroup>

## Get checkout embed details

Returns payment details plus a QR code URL for embedding checkout UI. Response `200`: `{ "data": { "id", "merchantAddress", "amount", "assetCode", "qrCodeUrl", "expiresAt" } }`

<CodeGroup>

```typescript SDK
const embed = await stellar.checkouts.retrieveEmbedDetails("cz_xxx");
```

```bash cURL
curl -X GET "https://stellartools.dev/api/checkout/cz_xxx/embed" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>

## Initiate payment (SEP-7)

Used by wallets to build the transaction. Body: `{ "account": "<customer_stellar_address>" }`. Response `200`: `{ "xdr", "network_passphrase", "callback": "url:..." }`

<CodeGroup>

```typescript SDK
const res = await fetch("https://stellartools.dev/api/checkout/cz_xxx/initiate", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ account: "G..." }),
});
const data = await res.json();
```

```bash cURL
curl -X POST "https://stellartools.dev/api/checkout/cz_xxx/initiate" \
  -H "Content-Type: application/json" \
  -d '{ "account": "G..." }'
```

</CodeGroup>
