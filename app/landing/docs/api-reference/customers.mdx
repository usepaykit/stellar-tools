title: 'Customers'
description: 'Create and list customers'

Customers represent payers. Create a customer to associate checkouts and subscriptions, and optionally attach Stellar wallets.

## Create customer

Name and email are required; phone, image, and metadata are optional. Image should be a public HTTPS URL. Response `200`: `{ "data": { "id", "name", "email", "phone", "metadata", "createdAt", "wallets": [] } }`

<CodeGroup>

```typescript SDK
const customer = await stellar.customers.create({
  name: "Jane Doe",
  email: "jane@example.com",
  phone: "+1234567890",
  image: "https://example.com/jane.png",
  metadata: { source: "web" },
});
```

```bash cURL
curl -X POST "https://stellartools.dev/api/customers" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane@example.com",
    "phone": "+1234567890",
    "image": "https://example.com/jane.png",
    "metadata": { "source": "web" }
  }'
```

</CodeGroup>

## List customers

Returns all customers for the organization. Optionally includes wallet addresses. Response `200`: `{ "data": [ { "id", "name", "email", "wallets", ... }, ... ] }`

<CodeGroup>

```typescript SDK
const customers = await stellar.customers.list({});
```

```bash cURL
curl -X GET "https://stellartools.dev/api/customers" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>

---

## Get customer

Response `200`: `{ "data": { customer object } }`

<CodeGroup>

```typescript SDK
const customer = await stellar.customers.retrieve("cust_xxx");
```

```bash cURL
curl -X GET "https://stellartools.dev/api/customers/cust_xxx" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>

## Update customer

All fields are optional; only include the ones you want to update. Response `200`: `{ "data": { updated customer } }`

<CodeGroup>

```typescript SDK
const updated = await stellar.customers.update("cust_xxx", {
  name: "Jane Smith",
  email: "jane.smith@example.com",
  phone: "+1234567890",
  image: "https://example.com/jane-smith.png",
  metadata: {},
});
```

```bash cURL
curl -X PUT "https://stellartools.dev/api/customers/cust_xxx" \
  -H "x-api-key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Smith",
    "email": "jane.smith@example.com",
    "phone": "+1234567890",
    "image": "https://example.com/jane-smith.png",
    "metadata": {}
  }'
```

</CodeGroup>

## Delete customer

Response `200`: `{ "data": null }`

<CodeGroup>

```typescript SDK
await stellar.customers.delete("cust_xxx");
```

```bash cURL
curl -X DELETE "https://stellartools.dev/api/customers/cust_xxx" \
  -H "x-api-key: your_api_key"
```

</CodeGroup>
