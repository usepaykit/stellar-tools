---
title: 'AI SDK Adapter'
description: 'Meter Vercel AI SDK usage with Stellar Tools'
---

The AI SDK adapter meters [Vercel AI SDK](https://sdk.vercel.ai) usage and charges Stellar Tools customers by token usage. It uses `createMeteredPlugin` from `@stellartools/plugin-sdk` for preflight, charge, and refund.

## Install

```bash
pnpm add @stellartools/aisdk-adapter ai
```

## Usage

Create a metered instance with `createMeteredAISDK(config)`. Each method takes `customerId` as the first argument, then the same arguments as the Vercel AI SDK (`generateText`, `generateObject`, `streamText`, `streamObject`).

```ts
import { createMeteredAISDK } from "@stellartools/aisdk-adapter";
import { openai } from "@ai-sdk/openai";

const metered = createMeteredAISDK({
  apiKey: process.env.STELLAR_TOOLS_API_KEY!,
  productId: "prod_ai",
});

// customerId first, then the same options as AI.generateText / streamText / etc.
const { text } = await metered.generateText("cust_xxx", {
  model: openai("gpt-4o"),
  messages: [{ role: "user", content: "Hello" }],
});

const result = await metered.generateObject("cust_xxx", {
  model: openai("gpt-4o"),
  schema: mySchema,
  prompt: "...",
});

// Streaming: preflight runs first; charge happens in onFinish
await metered.streamText("cust_xxx", {
  model: openai("gpt-4o"),
  messages: [{ role: "user", content: "Hello" }],
});

await metered.streamObject("cust_xxx", { model: openai("gpt-4o"), schema, prompt: "..." });
```

Methods:

- `generateText(customerId, ...args)` — same args as `AI.generateText`; meters by `result.usage.totalTokens`.
- `generateObject(customerId, ...args)` — same args as `AI.generateObject`; meters by usage.
- `streamText(customerId, options)` — preflight first; charges in `onFinish` by `event.usage.totalTokens`.
- `streamObject(customerId, options)` — preflight first; charges in `onFinish` by usage.

On insufficient credits the adapter throws; for `streamText`/`streamObject` it uses Vercel AI SDK’s `InvalidArgumentError` with `parameter: "credits"`.

## Config

`createMeteredAISDK(config)` expects `MeteredPluginConfig` from `@stellartools/plugin-sdk`:

- `apiKey`: Stellar Tools API key.
- `productId`: Credit product ID for AI usage billing.

Use a Stellar Tools credit product; if the customer has insufficient balance, preflight fails and the call does not run.
