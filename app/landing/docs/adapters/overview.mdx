Stellar Tools adapters plug Stellar payments and metering into your existing stack with minimal code.

## Available adapters

| Adapter | Use case | Package |
|---------|----------|---------|
| [MedusaJS](/docs/adapters/medusajs) | Payment provider for Medusa v2 (e-commerce) | `@stellartools/medusajs-adapter` |
| [Better Auth](/docs/adapters/betterauth) | Billing plugin: customers, subscriptions, refunds, credits | `@stellartools/betterauth-adapter` |
| [Uploadthing](/docs/adapters/uploadthing) | Metered file uploads (charge by bytes) | `@stellartools/uploadthing-adapter` |
| [LangChain](/docs/adapters/langchain) | Meter LLM usage (stream, batch, streamEvents) | `langchain-adapter` |
| [AI SDK](/docs/adapters/aisdk) | Meter Vercel AI SDK (streamText, streamObject) | `@stellartools/aisdk-adapter` |

## Core packages

- **@stellartools/core** — REST client for [Checkout](/docs/api-reference/checkout), [Customers](/docs/api-reference/customers), [Webhooks](/docs/api-reference/webhooks), [Subscriptions](/docs/api-reference/subscriptions), [Payments](/docs/api-reference/payments), [Refunds](/docs/api-reference/refunds), [Products](/docs/api-reference/products).
- **@stellartools/plugin-sdk** — Build metered integrations (preflight, charge, refund, meter). Used by Uploadthing, LangChain, and AI SDK adapters.
